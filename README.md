First clone the project

```bash
git clone https://github.com/HusseinKaraki99/weather-app.git 
```

Install node_modules

```bash
npm install 
```

Run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) .

- Axios
